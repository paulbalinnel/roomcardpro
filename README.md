# RoomPro Card

A premium, feature-rich custom Lovelace card for Home Assistant. Designed to emulate modern smart home interfaces with Glassmorphism, dynamic responsive grids, and internal popups.

## Features
- 🖼️ **Background Image & Glassmorphism:** Premium visual aesthetic.
- 📱 **Responsive Engine:** Uses CSS Container Queries. Auto-formats to single/double rows on iPad, wraps gracefully on mobile.
- 🎛️ **Internal Popups:** Dedicated modals for Audio control and Scene selection.
- 🎨 **Visual UI Editor:** Configure everything directly in HA.

## Installation via HACS
1. Ensure [HACS](https://hacs.xyz/) is installed.
2. Go to HACS -> Frontend -> Custom Repositories.
3. Add URL: `https://github.com/paulbalinnel/roomcardpro` (Category: Lovelace).
4. Search for "RoomPro Card" and install.
5. Refresh browser cache (Ctrl+F5).
